# Teardown production template — one video in ~20 minutes, faceless

**Format:** screen recording of a real store + captions (and/or AI voiceover).
No face, ever. Batch 5–7 at a time; post 1–2/day on TikTok + IG Reels + YT Shorts.

## Structure (30–45 seconds)

| Sec | Beat | Rule |
|-----|------|------|
| 0–3 | **Hook** (text on screen + spoken) | Name a $ number or a shocking specific. Never "let's review this store" |
| 3–8 | Show the store, orient fast | Scroll the homepage at 1.5–2x while stating the niche |
| 8–30 | **3 issues max**, worst first | For each: SHOW it (zoom/circle), say why it costs money, one-line fix |
| 30–40 | Score it | "This store: 54/100." (Score = instant format recognition + comment bait) |
| 40–45 | CTA | "Full 47-point audit — link in bio. Or comment AUDIT and I'll send you a free mini-teardown." |

## Production stack (all faceless)
- **Record:** Chrome + built-in screen record (or Screen Studio $89 for auto-zoom polish)
- **Captions:** CapCut auto-captions (free) — bold keywords, center-bottom
- **Voiceover:** your voice pitch-neutral OR ElevenLabs ($5/mo) — pick ONE voice and
  never change it (the voice becomes the brand)
- **Music:** trending, low volume, from the app's own library (better reach)

## Rules that make this work
1. **Real stores, anonymized-ish:** blur the store name/logo unless praising. Never
   punch down cruelly — "this store is SO close" beats "this store sucks."
2. **Specificity is the hook:** "$6.95 surprise shipping" outperforms "bad shipping UX."
3. **One niche per week** (candles week, skincare week…) — the algorithm learns who to
   show you to, and niche owners tag friends.
4. **Every teardown = a real checklist point** from `audit-engine/checklist.md`. Content
   IS the product demo.
5. **Comment "AUDIT" funnel:** reply to every AUDIT comment with the free mini-teardown
   DM → 3 issues → soft pitch the $297. This converts better than the bio link.
6. Volume over perfection: 30 posts/month. Expect 25 to do nothing. That's the model —
   the 5 that hit carry the account (that's the mechanic behind every "grew overnight"
   account people screenshot).

## Weekly cadence (~7 hrs/wk total)
- **Mon (2h):** pick niche, find 6 stores, run mini-audits (engine makes this fast)
- **Tue (2h):** record all 6
- **Wed (1h):** captions/edit/schedule via Meta Business Suite + TikTok scheduler
- **Daily (30m):** reply to comments, run the AUDIT DM funnel ← this is where sales happen
