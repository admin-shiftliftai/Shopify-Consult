# Shopify Consult — Faceless AI Shopify Audit playbook

Goal: $2k/mo within ~3 weeks via productized AI conversion audits → retainers → a
self-serve Shopify app at $10k+/mo. Solo, faceless, ~1–2 hrs/day after setup.
Entity: ShiftLift LLC (GA) — Stripe-ready today.

## The three phases
1. **Weeks 1–3 — first $2k:** sell $297–$997 AI-assisted audits, grown by faceless
   store-teardown short-form. ~4–7 audits = target.
2. **Weeks 3–8 — recurring:** convert audit clients to $750–$2k/mo optimization
   retainers; templatize until an audit takes <1 hr of human time.
3. **Months 3–6 — automated asset:** productize the audit engine into a Shopify App
   (Billing API + App Store distribution; sellable at ~3–5× ARR).

## What's in this repo
- `audit-engine/` — the product: 47-point checklist, 3-step prompt chain, report
  template, intake form, demo report, and a 15-min real-store dry-run runbook
- `offer/` — landing page copy + Stripe/delivery setup guide (~$50 total stack cost)
- `content-kit/` — teardown production template, 10 video scripts, 10 written posts
- `docs-run-first-audit.html` — visual, click-by-click beginner guide to running your
  first audit (open in any browser)
